# 多包名

## 场景

很多 Android 应用在开发时，会为了渠道运营等目的使用多个包名。

在这种场景下，可以使用云巴提供的 “多包名” 功能。

## 使用对象

**“多包名” 针对的对象是：同一个 [AppKey](product_kb_app_key.md)、不同包名的 Android 应用。**

## 如何使用

“多包名” 的使用很简单，只需在 [Portal](product_kb_portal.md) 创建应用时添加多个包名即可。也可以在应用创建后，根据需要添加新的包名。
